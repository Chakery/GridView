# GridView

A GridView that allows dragging the selection.

# Requirements
- iOS8.0+
- Xcode8.3+
- Swift3.0+

# Installation
## CocoaPods
```
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'
use_frameworks!

target '<Your Target Name>' do
    pod 'AyGridView', '~> 0.0.2'
end
```
